# 📊 Task 5 – Exploratory Data Analysis (EDA) using Python

## 📌 Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** performed on the **TechX Sales Dataset** using Python. The objective is to explore the dataset, understand its structure, identify patterns, detect anomalies, and generate meaningful business insights through statistical analysis and data visualization.

The analysis was completed using **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**, following industry-standard EDA practices.

---

## 🎯 Objectives

- Understand the dataset structure
- Perform statistical analysis
- Check missing and duplicate values
- Explore categorical and numerical features
- Identify relationships between variables
- Detect outliers and data distribution
- Generate business insights using visualizations
